# Personal Information

The Personal Information tab is the first section of the faculty dashboard. It contains two parts — your personal details and your addresses.

---

## Personal Details

| Field | Description | Required |
|---|---|---|
| Full Name | Your name as it appears across the system | Yes |
| Mobile | Your contact phone number | No |
| Date of Birth | Used for institutional records | No |
| Gender | Male / Female / Other | No |
| Nationality | Your nationality | No |
| Category | General / OBC / SC / ST / EWS | No |
| ORCID ID | Your ORCID researcher identifier (format: 0000-0000-0000-0000) | No |
| Profile Picture | JPG, JPEG or PNG, maximum 2MB | No |

### Saving Personal Details

Fill in the fields you want to update and click **Save Changes**. You do not need to fill in all fields at once — you can update individual fields at any time.

A green success message confirms the save. If there is an error, a red message is shown with the reason.

---

## Profile Picture

Profile pictures are uploaded directly to Cloudinary and stored as a URL in your profile. The image is automatically cropped to a 400×400 square with face detection.

### To upload a profile picture

1. Click the upload area below the current photo preview
2. Select a JPG, JPEG, or PNG file from your device
3. The upload happens immediately — no need to click Save Changes
4. The preview updates automatically once the upload is complete

:::info
The Photo URL field from the original profile has been replaced with a file upload. You no longer need to provide an external URL.
:::

:::warning
Files larger than 2MB will be rejected. Resize your image before uploading if needed.
:::

## Addresses

You can store two types of addresses:

| Type | Description |
|---|---|
| Correspondence Address | Your current mailing address |
| Permanent Address | Your permanent home address |

### Adding an Address

1. Click **+ Add** next to the address type you want to add
2. Fill in the address fields:
   - Address Line 1 (required) — house/flat number and street
   - Address Line 2 — area or locality
   - Address Line 3 — landmark (optional)
   - District (required)
   - State (required)
   - Country (required, defaults to India)
   - PIN Code (required)
3. Click **Save Address**

### Same as Correspondence Address

When adding your Permanent Address, if it is the same as your Correspondence Address, check the **Same as Correspondence Address** checkbox. All fields will be automatically filled from your correspondence address and disabled. Click **Save Address** to confirm.

:::info
You must add your Correspondence Address before using the "Same as" option for the Permanent Address.
:::

### Editing an Address

Click **Edit** next to the address you want to change. The form opens pre-filled with your existing data. Make your changes and click **Save Address**.

### Deleting an Address

Click **Delete** next to the address. You will be asked to confirm before the address is removed.
